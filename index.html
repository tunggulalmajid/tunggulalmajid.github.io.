ABOUT ME

Saya Tunggul Abdul Majid, Saya lahir di Kabupaten Banyuwangi pada tanggal 02 Februari 2006. Saya memiliki ketertarikan di bidang technology dan saya ingin menekuni bidang tersebut. saya memiliki cita cita sebagai mobile developer dan untuk mewujudkan cita cita saya itu, saya masuk ke dalam universitas jember terkhususnya ke fakultas ilmu komputer. meskipun saya belum memiliki dasar untuk mewujudkan cita cita saya karena saya berasal dari SMA, saya akan lebih berusaha untuk mengejar ketertinggalan saya sehingga nantinya saya dapat menggapai cita cita saya yaitu sebagai mobile developer. Kenapa saya memilih mobile development sebagai cita cita saya ?. saya memilih mobile development karena menurut saya mobile development merupakan sesuatu yang menarik, ditambah lagi pengembangan software khusunya software untuk mobile sangat berkembang pesat sehingga terdapat peluang besar untuk kedepannya.
